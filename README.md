# LHSP - Luan's Homebrew'd Switch Pack
This is an Pack for Homebrew'd switches for easy setup, its simple and lightweight.

# Current Installed Features
- HEKATE 6.5.1
- ATMOSPHERE 1.10.2
- SIGPATCHES (sys-patch) 
- UltraHand overlay & Nx-Ovlloader
- SYS-CLK
- DBI
- GOLDLEAF

# Exosphere Warning
Your EmuMMC WILL have it's serial and prod info blanked, so you wont get banned if you delete or change the hosts folder on atmosphere. but if you want you can disable the flag, go to /exosphere.ini.
You have been warned of a potential ban if you do so or get your serial info stolen by an cracker.
